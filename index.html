Finding Twitter symmetric social relations
Finding Twitter symmetric social relations

Relations in twitter are not necessarily symmetric relations. User A may follow user B while user B may or may not follow user A. The goal of this project was to find all pairs of users A, B such that A follows B and B follows A.

Input dataset for the project was a large text file of Twitter's social graph that contains over a billion social relations between twitter users. The size of the dataset was 25 GB. Developed a MapReduce job to extract such users A, and users B that follow each other. Executed the MapReduce locally first on a sample dataset and finally on Amazon Web Service using the Amazon Elastic MapReduce.
